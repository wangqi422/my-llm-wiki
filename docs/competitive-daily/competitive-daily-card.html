Design Reference

设计参考价值

1

三角洲行动 × 古墓丽影联动视觉

跨 IP 联动是当前 FPS 品类重要宣发手段，劳拉角色融入军事射击场景的视觉融合方案值得关注，可参考其海报风格处理 IP 角色与原生世界观的视觉统一性。

2

三角洲行动前瞻直播视觉包装

赛季主题「回声」围绕声波 / 水下作战核心，采用多栖环境的视觉语言，蓝绿色调 + 声波纹理的设计方向可供 CODM 赛季 KV 参考。

3

Battlefield 6 Season 视觉体系

Hunter/Prey 主题的狩猎者 vs 猎物对立视觉叙事，Season 更迭的视觉连续性处理手法值得借鉴。
