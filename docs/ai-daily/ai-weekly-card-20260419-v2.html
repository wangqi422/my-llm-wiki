8 CLAUDE.md 项目契约 · 让 Claude Code 2.1 真正懂你 Best Practice
4月14日 Claude Code 2.1.101 官方最新 Best Practices：项目根目录的 CLAUDE.md 文件，比任何花哨 Prompt 都管用。同一个模型，有没有 CLAUDE.md 的差距 = junior vs senior，返工率下降 60%+。
# Project: [项目名] ## 🎯 目标 [一句话：这个项目做什么] ## 🛠 技术栈 - 语言：[Python 3.12 / TS 5.x] - 包管理：[uv / pnpm] ## ✅ 必须遵守 - 所有新函数必须带类型标注 - 外部调用必须有超时 + 重试 - Commit 格式：<type>(<scope>): <subject> ## ❌ 禁止事项 - 不要直接改 migrations/ 下的历史文件 - 不要在 src/ 里写一次性脚本 ## 🧪 验收标准 1. make test 全绿 2. make lint 无新增 warning 3. 新增功能有对应测试
Source: @AnthropicAI, Claude Code Docs 2.1.101
