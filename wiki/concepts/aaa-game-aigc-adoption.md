# AAA Game AIGC Adoption

> 3A游戏工作室公开采用AIGC制作游戏资产的行业趋势、标杆案例与风险管控。

## Definition

指3A级游戏IP在开发和宣发过程中正式、公开地使用AI生成式工具创作游戏内资产和营销素材。2025年Activision在《使命召唤：黑色行动6》Steam页面声明使用AIGC，成为行业标志性事件。

## Core Content

### Activision/COD案例
- 在BO6和Warzone的Steam页面正式声明使用AI工具
- 涉及领域：概念艺术、武器皮肤、加载屏幕、环境纹理
- 响应Steam AI透明度规则（2024年1月生效）
- 社区对"六指圣诞僵尸"等AI痕迹表达担忧

### 风险管控三段式流程
```
AI生成 → 人工精修 → QA验收
```

### 需要关注的风险
- AI痕迹被玩家识别引发舆论风险
- 手指数量、文字变形等细节瑕疵
- 国内可能出台的AI内容标注规则
- 艺术家就业影响的社区讨论

## Key Points

- Activision是3A游戏IP首次公开大规模使用AIGC
- 微软正加大AI游戏开发投入（Muse工具等）
- 对CODM团队：既是"行业背书"，也需要注意质量风险
- 建立AI素材QA Checklist是当务之急
