# AI Video Production Strategy

> 从双引擎到三层架构的AI视频分级生产体系，配套行业首套质量三支柱标准。

## 三层分级生产架构

从04.09日报的"双引擎"演进为04.11的"三层架构"：

```
Layer 1 - 品牌级    → Runway Gen-4.5（世界模型/物理模拟/最高品质）
Layer 2 - 铺量级    → Kling 3.0（$0.07/秒/多镜头/统一架构/性价比王）
Layer 3 - 自部署级  → HappyHorse（开源/数据安全/零API成本）
验证层   - 免费验证  → Google Veo 3.1（创意快速探索/零成本）
```

### 双引擎核心对比

| 维度 | [[runway-gen4]] (品牌级) | [[kling-3]] (铺量级) |
|------|--------------------------|----------------------|
| 核心优势 | 角色跨场景一致性最佳 | 性价比之王 $0.07/秒 |
| 特色功能 | Act-Two动捕、Aleph编辑 | 多镜头序列(2-6场景)、原生音频 |
| 适用场景 | 品牌CG、官方预告片 | 社媒短视频、批量素材 |
| 成本 | 基准 | 约42%（Runway的42%） |

### 内容分级

```
A级素材（品牌宣传片/CG预告）→ Layer 1: Runway Gen-4.5
B级素材（社媒视频/活动素材）→ Layer 2: Kling 3.0
C级素材（测试素材/内部预览）→ Layer 3: HappyHorse / 免费验证层
```

## 质量三大支柱（行业标准）

1. **技术卓越与保真度**：AI视频在随意查看下无法与真实素材区分
2. **真实性与出处**：强制元数据标记、数字水印、区块链出处追踪
3. **创意控制与迭代**：支持协作而非仅生成，人类艺术指导融合

### 质量评判新定义

从分辨率和制作价值 → **叙事完整性 + 情感共鸣 + 受众连接**

### 三级审查体系

```
第一级：技术质量（分辨率/连贯性/运动自然度）
第二级：品牌合规（色彩/角色/视觉规范）
第三级：叙事品质（情感传达/受众连接）
```

## 策略张力：统一架构 vs 分层组合

- [[kling-3]] 统一架构："单平台覆盖全流程"——零上下文切换，批量效率最高
- 三层分级架构："多工具精细控制"——风险分散，各层最优
- **CODM需A/B测试确定最优路径**

## CODM Application

- **分级匹配**：按内容级别（A/B/C）选择对应工具层
- **质量Checklist**：参考三支柱建立CODM AI素材质量标准
- **三级审查SOP**：技术→品牌→叙事
- **成本优化**：AI视频成本下降91%，预算从制作转向创意
