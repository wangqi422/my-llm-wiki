# CODM PPT Visual Style — 暗黑军事科技风格规范

> CODM（使命召唤手游）内部演示文稿的统一视觉设计语言：以深色背景+荧光黄点缀+军事角色衬底为核心的硬核科技风。

## Definition

CODM PPT Visual Style 是 CODM 宣发设计团队在制作内部分享、方案汇报、AI工作流演示等演示文稿时遵循的一套视觉设计规范。该风格深度融合了 Call of Duty 游戏IP的军事硬核调性与现代科技感，形成了极具辨识度的品牌演示语言。

## Core Content

### 1. 色彩体系 (Color Palette)

| 角色 | 颜色 | HEX | 使用场景 |
|------|------|-----|----------|
| **主背景色** | 极深灰/近黑 | `#151619` ~ `#23262A` | 幻灯片背景（通过深色角色图叠加实现） |
| **强调色/点缀色** | 荧光黄 | `#FCFF34` | 装饰线条、章节分隔条、几何装饰元素、导航箭头 |
| **主文字色** | 白色 | `#FFFFFF` / `bg1` | 标题、正文文字（在深色背景上） |
| **次要文字色** | 中灰色 | `#575757` | 辅助说明、层级较低的文字 |
| **暗文字色** | 深灰 | `#23262A` | 年份标注、侧边"Call of Duty mobile"水印 |

**色彩哲学**：整体色调极度克制——深色系占比 95% 以上，仅通过荧光黄 `#FCFF34` 形成视觉焦点。这与 COD 系列游戏中"夜战""潜行"的主题高度一致，同时荧光黄类似战术HUD显示器的光标色，传达出军事科技感。

### 2. 字体体系 (Typography)

| 层级 | 字体 | 字号范围 | 特征 |
|------|------|---------|------|
| **主标题** | 微软雅黑（Microsoft YaHei） | 44-60pt | 加粗(bold)、居中对齐 |
| **副标题/章节标题** | 微软雅黑 | 20-44pt | 加粗、居中或左对齐 |
| **正文/描述** | 微软雅黑 | 8-16pt | 常规或加粗 |
| **侧边水印** | 微软雅黑 | 8pt | 全大写(cap="all")、宽字距(spc="1000") |
| **辅助文字** | 微软雅黑 | 6.5-8pt | 用于页码、版权标注 |

**字体哲学**：统一使用微软雅黑作为唯一字体族，通过**字重（粗/细）+字号（极大/极小）+字距（紧密/宽松）**三个维度制造层级差异，而非依赖多种字体。英文副标如"Call of Duty mobile"采用全大写+超宽字距(letter-spacing: 1000)的处理方式，呈现出军事编码/战术代号的视觉感。

### 3. 背景系统 (Background System)

CODM PPT 使用**三层背景叠加**实现深沉的视觉氛围：

1. **底层**：纯深色（`#151619`近纯黑）
2. **中层**：CODM游戏角色（Ghost等标志性人物）的高分辨率暗化图——角色位于画面左侧或中部，整体亮度压暗至 10-20%，形成"隐约可见"的幽灵效果
3. **顶层**：还叠加了带有数码扫描线(scan lines)的纹理效果，模拟军事监控画面

**交替使用两套背景**：
- **A套**（slide 5, 8, 11, 14, 17...）：较亮的角色背景（image6.png），搭配荧光黄底部装饰条
- **B套**（slide 6, 9, 12, 15...）：更暗的纯色背景（image8.png），背景几乎全黑

### 4. 装饰元素 (Decorative Elements)

- **装饰条**（image5.png）：斜切的灰色锯齿线条，模仿军事技术图纸中的剖面线，以极低透明度（alpha=10%）横置于幻灯片上方
- **荧光黄几何元素**：
  - 矩形色块 + 带"V形缺口"的自定义多边形（如章节标号旁的战术箭头）
  - 底部进度条/分隔条
  - 小方块装饰点（左上角、右下角，类似HUD元素）
- **CODM Logo**（image4.png）：灰色半透明 CALL OF DUTY 使命召唤手游 Logo，位于幻灯片右上角
- **团队标识**（image3.png）："J3宣发设计组" 标识，位于幻灯片左下角

### 5. 布局系统 (Layout System)

**五种核心幻灯片类型**：

| 类型 | 布局特征 | 使用场景 |
|------|---------|----------|
| **封面页** | 居中标题 + CODM Logo + 底部水印 + 侧边旋转文字 | 第1页 |
| **目录页** | 居中编号 + 多行分类 + 缩略图预览 | 第3页 |
| **章节过渡页** | 大章节号（如"1.1"）+ 标题 + 装饰形状 + 全幅背景 | 第4、10、20、25、33、36页 |
| **内容页** | 左上角小标题 + 大面积图片展示 + INPUT/OUTPUT对比 + 流程节点标注 | 占比最多 |
| **效能对比页** | 时间线对比（旧流程 vs AI流程）+ 时间节点标注 | 第9、19页 |

**布局原则**：
- 侧边栏水印"Call of Duty mobile"始终**旋转90度**置于左侧（rotation=16200000 = 270°）
- 右上角始终有CODM Logo
- 左下角始终有团队Logo
- 内容区域采用大面积图片展示（尤其是AI生成效果的before/after对比）
- 流程展示使用"应用工具 → 流程节点 → 图片输入/输出"的三段式结构

### 6. 文案风格 (Copy Style)

- **幻灯片标题**：简短有力，如"骨架与光影"、"资产棋局"、"质感跃迁"
- **章节引言**（出现在"目标"页）：诗意化/格言化的总结，如：
  - "没有高级资产，那是靠自己造。在抠像出血的过程中，有条有据地用住 AI 的手，让资产的精雕细实现。"
  - "精雕不是磨难，而是花园；AI 是放大镜的脚蹬师，赋能在放大细节的同时，不要忽略和推倒了一切全域性的底骨。"
- **版权声明**：封面标注"此PPT不是AI教学，只在分享制作方法"
- **章节编号**：采用"1.1""1.2""1.3"层级编号，配合页码标注（如"PG.06"）

## Key Points

- **极致暗色调**：95%以上面积为深色(#151619~#23262A)，对比度极高，确保内容在投影环境中仍有冲击力
- **荧光黄唯一点缀色**：`#FCFF34` 是整个模板中唯一的彩色元素，用于装饰线条和几何图形，不用于文字
- **游戏IP深度融入**：背景使用暗化的游戏角色图，而非抽象图案，强化了品牌归属感
- **军事HUD美学**：扫描线纹理、全大写宽字距文字、战术箭头几何元素、HUD小方块装饰——整体模拟军事科技界面
- **统一字体族**：全局使用微软雅黑，通过字重/字号/字距变化制造层级
- **大面积留白（留黑）**：深色背景上留出大量暗色空间，让少量高亮内容更加突出
- **before/after对比范式**：内容页核心是INPUT→OUTPUT的对比展示，天然适合AI工作流的效果呈现

## CODM Application

此PPT风格是CODM宣发设计团队的官方演示标准模板，直接应用于：
1. **内部AI工作流分享**（如本PPT"视觉中枢重塑3.17"）
2. **版本设计方案汇报**
3. **竞品分析演示**
4. **跨部门技术分享**

在使用AI制作PPT或演示材料时，应严格遵循此风格规范，确保品牌视觉一致性。与 [[game-version-visual-guide]] 中定义的游戏内视觉指引形成互补——游戏视觉指引管控游戏内宣发素材风格，本PPT风格管控对外/对内演示的文档风格。
