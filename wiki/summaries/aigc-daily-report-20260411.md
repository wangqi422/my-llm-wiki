# AIGC 宣发设计日报 2026-04-11

> 覆盖游戏美术、写实AIGC、视频生成、AI Agent四大领域的每日精选 TOP 10，服务于 CODM 宣发设计团队。

## Brief

本期日报聚焦过去24小时（2026-04-10 ~ 2026-04-11）的行业动态。核心发现：**开源AI视频模型首次击败所有闭源巨头**（HappyHorse-1.0登顶Artificial Analysis）、**品牌定制AI赛道爆发**（Stability Brand Studio + Adobe Firefly Custom公测 + qwen-trainer三方竞争）、**AI视频生成加速"基础设施化"**（Veo 3.1免费+Sora确认4.26关停）、**DALL·E 3即将停服**（GPT Image 1.5迁移窗口仅剩30天）。

## Key Takeaways

1. **开源力量颠覆格局**：[[happyhorse-1]] 以150亿参数开源模型在AI视频排行榜双项登顶，打破闭源垄断，为自部署提供可能
2. **品牌定制AI三足鼎立**：[[stability-brand-studio]] + [[adobe-firefly]] Custom + [[qwen-image-trainer]] 形成三方竞争，品牌AI从"奢侈品"变为"标配"
3. **AI视频"基础设施化"加速**：[[google-veo-3]] 免费开放 + HappyHorse开源，AI视频生成正从专业工具变为基础设施
4. **OpenAI生态巨变**：DALL·E 3将于5.12停服，[[gpt-image-1-5]] 成为新旗舰，文本渲染和Logo保留能力大幅提升
5. **双引擎→三层架构演进**：从 [[ai-video-production-strategy]] 演进为"品牌级+铺量级+自部署级"三层视频生产架构
