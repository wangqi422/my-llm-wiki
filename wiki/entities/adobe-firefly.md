# Adobe Firefly

> 集成30+顶级模型、支持品牌自定义模型训练的综合AI创意平台。

## Info

| Field | Value |
|-------|-------|
| Type | tool / AI creative platform |
| Domain | AI图像/视频创意 |
| Developer | Adobe |
| Website | https://firefly.adobe.com |

## Overview

Adobe Firefly 推出自定义模型训练功能（公测中），用户可上传品牌视觉资产训练专属模型。集成30+顶级模型（含Google Nano Banana 2、[[runway-gen4]] Gen-4.5、[[kling-3]] 2.5 Turbo等），成为"A模型生成→B模型优化→Adobe工具精修"的集成平台。Project Moonlight 对话式AI创意代理标志"提示词驱动"到"对话式协作"的转变。是 [[brand-custom-ai-model]] 概念的核心实现。

## Key Contributions

- 自定义模型训练（品牌资产→专属模型）
- 30+顶级模型集成平台
- Project Moonlight 对话式AI创意代理
- 与PS/AE/PR全套Adobe工具深度整合
- **[2026-04-11]** 自定义模型正式进入公测，50-100张参考图即可训练，无需ML背景
- **[2026-04-11]** 训练结果PS/AI中直接调用，与设计师工具链无缝衔接
- **[2026-04-11]** 与 [[stability-brand-studio]] / [[qwen-image-trainer]] 形成品牌定制AI三方竞争
