## Overview

Lovart 是由LiblibAI推出的全球首个专业设计Agent，与传统AI图像生成工具不同，它不是"生成单张图片"，而是"理解整个设计项目"——从需求理解、方案规划、多模态生成（图像/视频/3D）到一致性编辑和导出的全链路自动化。

核心能力包括：Edit Elements分层图像编辑、4K超清输出、品牌风格一致性保持、图文海报一键生成、批量尺寸适配。集成GPT image-1/Flux Pro/OpenAI o3等多种前沿模型，支持LOGO/社媒图形/营销活动全套物料。

被设计社区评价为"第一个真正理解设计工作流的AI"——它是"设计师同事"而非"生图工具"。

## Key Contributions

- 全球首个"设计Agent"——从单点生图到全链路设计自动化
- 需求理解→方案规划→生成→编辑→交付的完整工作流
- Edit Elements分层编辑（单图层精细调整）
- 品牌风格一致性引擎
- 批量尺寸/平台适配
- 多模型集成（GPT image-1 + Flux Pro + o3）

## CODM Application

- **全套物料一键生成**：输入赛季主题brief，自动输出KV+社媒多尺寸+Banner+视频封面全套
- **品牌一致性引擎**：确保从主KV到最小社媒缩略图都保持CODM视觉DNA
- **分层编辑能力**：AI生成后可对单个图层进行精细调整，兼顾效率和品质
- **跨平台适配**：一次生成自动适配抖音/TikTok/Reels/微博/微信等多平台尺寸
- ⚠️ **待验证**：品牌还原精度、中文设计理解能力、与现有PS/Figma工作流的兼容性
