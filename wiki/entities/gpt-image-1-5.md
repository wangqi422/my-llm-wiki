# GPT Image 1.5

> OpenAI最新旗舰图像生成模型，在指令遵循、文本渲染和Logo保留方面大幅超越DALL·E 3。

## Info

| Field | Value |

| API Cost | 比前代降低20% |

## Overview

GPT Image 1.5是OpenAI取代DALL·E系列的新旗舰图像模型，不再是独立扩散模型而是与GPT系统深度融合的"理解型"图像引擎。在Microsoft内部测试中，prompt对齐度和信息图任务均超越其他模型。核心突破包括指令遵循能力领先、文本渲染清晰度大幅提升、编辑时Logo/人脸保留一致性改善、生成速度提升4倍。

随着DALL·E 2/3将于2026年5月12日停服，GPT Image 1.5成为OpenAI生态唯一的图像生成选择。

## Timeline

- **2025-12**: GPT Image 1.5发布
- **2026-04-06**: DALL·E 3停服迁移指南发布
- **2026-05-12**: DALL·E 2/3正式停服截止日

## Key Contributions

- 指令遵循能力业界领先（prompt对齐度+信息图任务双冠）
- 文本渲染质量大幅提升，适合营销banner和海报
- 编辑一致性：Logo/人脸/构图保留能力强
- 生成速度4x提升，API成本降低20%
- 支持透明背景输出和多格式导出

## CODM Application

- Logo保留能力：在AI生成场景中精确嵌入CODM标识
- 文本渲染：直接在生成图中添加活动名称/Slogan
- 信息图制作：赛季统计、数据可视化等复杂布局
- ⚠️ **紧迫**：如有DALL·E 3依赖，须在5月12日前完成迁移
