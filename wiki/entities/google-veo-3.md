# Google Veo 3.1

> Google免费开放的AI视频生成模型，标志AI视频"基础设施化"的里程碑。

## Info

| Field | Value |
|-------|-------|
| Type | tool / AI video platform |
| Domain | AI视频生成 |
| Developer | Google DeepMind |
| Access | Google Vids / Google Flow |
| Free Tier | 是（每月10视频 / 每日50积分） |
| Paid | AI Ultra $249.99/月（1080p+无水印） |

## Overview

Google于2026年4月2日向所有Google账户免费开放Veo 3.1视频生成模型。免费版可通过Google Vids（每月10个视频）和Google Flow（每日50积分/约10-12个Lite片段）使用。支持文本生视频和图片生视频，原生竖屏(9:16)输出。免费版限制为720p、8秒/片段、含水印。

这是AI视频生成从"专业工具"到"基础设施"的标志性转折点。

## Timeline

- **2026-04-02**: Google宣布Veo 3.1免费开放
- **2026-04-07**: Google Vids集成Veo 3.1，支持免费视频生成+YouTube导出

## Key Contributions

- 首个顶级AI视频模型面向所有用户免费
- 原生竖屏（9:16）输出适配短视频平台
- Google Vids集成实现生成→编辑→YouTube发布全链路
- 推动AI视频从专业工具走向基础设施

## Technical Specs (Free Tier)

| Spec | Value |
|------|-------|
| 最高分辨率 | 720p (免费) / 1080p-4K (Ultra) |
| 单片段时长 | 最长8秒 |
| Google Vids额度 | 每月10个视频 |
| Google Flow额度 | 每日50积分（Lite≈10-12片段/Fast≈3-5片段） |
| 水印 | 免费版含"Made with Veo"水印 |
| 音频 | Google Vids暂不支持音频生成 |

## CODM Application

- 免费版作为"创意探索层"，每天测试10+个创意方向
- 原生竖屏输出直接适配抖音/TikTok/YouTube Shorts
- 定位为三层视频架构中的"验证层"：Veo验证→Kling/Runway制作→人工精修
- ⚠️ **限制**：720p+水印+无音频，不适合最终交付素材
