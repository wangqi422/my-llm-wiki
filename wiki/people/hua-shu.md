# 花叔（AlchainHust）

## 简介

AI Native Coder、独立开发者、AI 自媒体博主（30 万+ 粉丝）。代表作「小猫补光灯」曾登 App Store 付费榜 Top 1。知识星球「AI 编程：从入门到精通」主理人。现居深圳。

## 关键贡献

- **橙皮书系列**：编写了多本免费开源电子书，包括《Claude Code 从入门到精通》、《Claude Code 源码解析》、《Agent Skills》、《Harness Engineering》、《Obsidian + Claude Code：用 AI 重建你的第二大脑》等 — [[raw/articles/obsidian-ai-orange-book-full.md]]
- **CLAUDE.md 路由器体系**：在一年多的实践中演化出 9 个 [[wiki/concepts/claude-md.md|CLAUDE.md]] 组成的路由系统，根目录 CLAUDE.md 根据关键词把任务分发到不同工作区 — [[raw/articles/obsidian-ai-orange-book-full.md]]
- **55 个 [[wiki/concepts/harness-engineering.md|Skills]]**：分三类——人物视角（20+ 思想者蒸馏，项目「女娲」GitHub 6000+ 星）、内容创作（选题→三审三校→发布全链路）、工具类（PDF/视频/飞书/图床）— [[raw/articles/obsidian-ai-orange-book-full.md]]
- **一人自媒体运营**：靠 CLAUDE.md + Skills 体系一人撑起多平台内容运营（公众号、B 站、X/Twitter、YouTube、小红书）— [[raw/articles/obsidian-ai-orange-book-full.md]]
- **2000+ Markdown 文件**的写作知识库，用 Cursor 文件树 + Claude Code 管理，后迁移至 [[wiki/concepts/obsidian.md|Obsidian]] — [[raw/articles/obsidian-ai-orange-book-full.md]]
