## 1.1 战斗与交互特效

### 1.1.1 枪口焰与抛壳

**枪口焰三阶段生命周期模型：**

```
时间轴（30fps）
──────────────────────────────────────────────────────►

Phase 1: ██ 爆发期 (1-2帧)
         核心火焰 + 环境光照   → 视觉冲击力峰值

Phase 2:  █████ 喷射与扩散期 (2-5帧)
          火花 + 冲击波 + 高速烟  → 速度感 + 粒子动态

Phase 3:   ████████████ 消散与残留期 (10+帧)
            滞留烟雾 + 热扭曲 + 枪管余烟  → 真实感 + 空气密度
```

| 狙击枪 | 长管抑制/大口径爆闪 | 高 / 长条 | 消音无/无消大量 | 大口径手动 | 2-3m |

# 02. 制作管线

## 2.1 枪口焰制作全流程

```
Phase 1: 参考收集
├─ 实拍高速摄影素材 (1000-10000fps)
├─ ActionVFX等素材库高清参考
└─ 不同枪型/口径/环境下的差异记录
        ↓
Phase 2: 概念设计
├─ 2D概念帧（形状、颜色、规模）
├─ 时序节奏设计（三阶段时长）
└─ 交互光照范围设计
        ↓
Phase 3: 3D制作 (三条技术路线)
│  方案A: Houdini Pyro → VDB体积
│  方案B: 多层粒子系统 (Maya/UE/Unity)
│  方案C: Flipbook序列 (Houdini→Sprite Sheet)
        ↓
Phase 4: 光照交互
├─ 伴随动态点光源
├─ 颜色随时间插值 (白热→橙黄→暗红)
└─ 环境阴影投射
        ↓
Phase 5: 渲染
├─ 离线: Arnold/V-Ray/Redshift
└─ 分层: 火焰层/烟雾层/光照层
        ↓
Phase 6: 合成 (Nuke/AE)
├─ 分层合成 + Bloom/Glow/Lens Flare
└─ 与背景融合调色 → 最终输出
```

**各公司枪火技术路线对比：**

| 公司 | 软件栈 | 方案 | 特点 |
|------|--------|------|------|
| Blur Studio | 3ds Max+XSI+自研 | Houdini Pyro→3ds Max合成 | 电影级，单帧渲染时间长 |
| Digic | Maya+Houdini+内部管线 | Houdini模拟+Maya粒子混合 | 大规模多枪优化 |
| Platige | Maya+Houdini+Nuke | Houdini全流程 | 高质量体积渲染+电影级合成 |
| 游戏引擎内 | 自研引擎 | GPU粒子+Flipbook+动态光源 | 实时性能优先 |

## 2.2 爆炸制作全流程

```
Step 1: 参考与设计 → 确定爆炸类型/规模/分层概念图
        ↓
Step 2: 破坏模拟 (Houdini RBD)
        原始模型→Voronoi碎片化→约束→爆炸力场→碎片物理→Alembic/VDB
        ↓
Step 3: 火焰模拟 (Houdini Pyro)
        爆炸源→速度场→Temperature/Fuel/Density/Turbulence/Dissipation调节
        ↓
Step 4: 烟雾模拟 (独立二次Pyro)
        热浮力驱动上升 + 风场影响
        ↓
Step 5: 碎片粒子 → 尾迹烟雾 + 落地二级尘土
        ↓
Step 6: 冲击波 → 球形扭曲 + 环形扬尘
        ↓
Step 7: 分层渲染 (Fire/Smoke/Debris/Dust/Light/Shadow各层独立输出)
        ↓
Step 8: 合成 (Nuke) → Bloom/Lens Flare/Heat Distortion + 调色 + 摄像机震动
```

**各公司爆炸技术差异：**

| 公司 | 模拟工具 | 渲染器 | 特色 |
|------|---------|--------|------|
| ILM | Houdini+自研Plume | Arnold | 《变形金刚》级超大规模 |
| Weta | Houdini+自研Synapse | Mantra | 战场级连环爆炸 |
| DNEG | Houdini | Clarisse/Arnold | 《信条》物理逆转爆炸 |
| Blur | Houdini→3ds Max | V-Ray | 游戏CG级精致爆炸 |
| Platige | Houdini | Arnold | COD冷战双重氛围爆炸 |

## 2.3 子弹命中交互制作流程

```
Step 1: 材质定义 → 每个物体标记物理材质标签
        ↓
Step 2: 碰撞检测 → 获取命中点/法线/角度/材质
        ↓
Step 3: 按材质分支触发特效
        混凝土 → 粉尘爆射+碎块弹射+粉尘云+弹坑Decal
        金属   → 高速火花+金属碎屑+凹痕Decal
        木材   → 木屑+纤维+弹孔Decal (大口径触发预断裂)
        (玻璃/泥土/雪/水/人体各自分支...)
        ↓
Step 4: Decal贴花 → 角度投射+随机化+累积+远距消除
        ↓
Step 5: 音效触发 → 材质标签→音效库+距离衰减+混响
```

## 2.4 CG预渲染 vs 实时引擎对比

| 维度 | CG预渲染 | 实时引擎 |
|------|---------|---------|
| 模拟精度 | Houdini高精度流体 | GPU粒子+Flipbook |
| 碎片数量 | 数千-数万 | 数十-数百 |
| 烟尘 | 体积渲染VDB | Billboard粒子近似 |
| 光照 | 全局光照+光追 | 简化动态点光源 |
| 渲染时间 | 每帧数分钟-数小时 | 每帧<16ms(60fps) |
| 工具 | Houdini+Maya+Nuke | UE5 Niagara / Unity VFX Graph |

## 2.5 核心软件工具栈

**离线制作（CG预告片）：** Maya/3ds Max/ZBrush(建模) → Maya/MotionBuilder(动画) → **Houdini(特效行业标准)** → Arnold/V-Ray/Redshift(渲染) → **Nuke(合成行业标准)** → Premiere/DaVinci(剪辑)

**实时制作（游戏内）：** UE5 Niagara/Unity VFX Graph(粒子) → Houdini→Sprite Sheet(Flipbook烘焙) → HLSL/ShaderGraph(Shader) → 引擎Post-Processing(后处理)

## 4.1 七维度分析框架

| 维度 | 关注点 | 记录内容 |
|------|--------|---------|

| **7. 后处理** | 调色/DOF/Bloom 等 | 色调冷暖/景深焦点变化 |
