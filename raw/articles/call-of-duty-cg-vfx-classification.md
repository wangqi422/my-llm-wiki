## 一、光照与阴影特效

光照是 CG 画面质感的基石。使命召唤 CG 中的光照特效涵盖以下子类：

### 1.1 全局光照（Global Illumination）

| 技术 | 说明 | 应用场景 |
|------|------|---------|
| **SDF-GI（有符号距离场全局光照）** | IW 9.0 引擎最新方案，实时计算间接光反弹 | 室内场景的柔和间接光、走廊中的光线扩散 |
| **预计算体积光照** | Activision Research 发表的压缩方案，受限球谐函数表示 | 大规模开放地图的全局光照 |
| **光探针（Light Probes）** | 场景中布设采样点，插值计算间接光 | 角色在不同环境下的自然光照融入 |
| **屏幕空间全局光照（SSGI）** | 利用屏幕深度信息估算间接光 | 近距离物体间的颜色渗透效果 |

### 1.2 直接光照

| 技术 | 说明 | 应用场景 |
|------|------|---------|
| **动态聚光灯/点光源** | 实时可移动光源，支持衰减和颜色控制 | 手电筒、枪口焰照亮周围、爆炸闪光 |
| **方向光（太阳/月光）** | 场景主光源，控制整体光照氛围 | 白天战场、夜间月光、阴天灰暗氛围 |
| **区域光** | 模拟柔和面光源 | 窗户漏进的柔光、荧光灯管照明 |

### 1.3 光线追踪

| 技术 | 说明 | CG 表现 |
|------|------|---------|
| **光追阴影（RT Shadows）** | 像素级精确的柔和阴影 | 树叶投影、围栏镂空影子、砖缝阴影 |
| **光追反射（RT Reflections）** | 金属/水面/玻璃的物理正确镜面反射 | 雨后地面倒影、枪械金属反光、车窗反射 |
| **光追环境光遮蔽（RT AO）** | 替代传统 SSAO 的高精度遮蔽 | 角色与地面接触处的接触阴影 |

### 1.4 体积光照效果

| 技术 | 说明 | CG 表现 |
|------|------|---------|
| **体积雾光（Volumetric Fog/God Rays）** | 光线穿透烟雾/灰尘的丁达尔效应 | 窗户射入的光柱、爆炸后烟尘中的光束 |
| **体积云** | 三维云层渲染 | 空战场景、高空跳伞序列 |
| **大气散射** | 模拟大气对远处光线的散射 | 远景山峦蓝调、日出日落橙红天空 |

### 1.5 阴影系统

| 技术 | 说明 | CG 表现 |
|------|------|---------|
| **级联阴影贴图（CSM）** | 多级分辨率阴影贴图 | 远近不同精度的地面阴影 |
| **点阴影缓冲** | 预加载点光源阴影到显存 | 室内灯光的实时阴影无延迟弹出 |
| **自阴影（Self Shadow）** | 材质法线生成的微观自遮蔽 | 砖墙缝隙、织物褶皱的微阴影 |
| **接触阴影（Contact Shadows）** | 屏幕空间的近距离小尺度阴影 | 枪械贴近身体时的柔和阴影 |

## 二、粒子与流体特效

粒子系统是使命召唤 CG 中**占比最大、类别最丰富**的特效类型。

### 2.1 爆炸系统（多层复合特效）

一次完整的 CG 爆炸由 **5+ 层粒子子系统** 叠加而成：

```
时间轴 ──────────────────────────────────────────────►

Layer 1: ████ 闪光核心（Flash Core）
         极亮白/橙色球体，持续 2-3 帧

Layer 2:  ██████████ 火球（Fireball）
          橙红色翻滚火焰体积，持续 0.5-1s

Layer 3:   ████████████████ 冲击波（Shockwave）
           环形扩散的空气扰动波纹，持续 0.3-0.5s

Layer 4:    ████████████████████ 碎片飞溅（Debris）
            混凝土块、金属碎片、石子弹射

Layer 5:     ██████████████████████████████ 烟尘扩散（Smoke Plume）
             浓黑/灰色烟柱上升扩散，持续 3-10s

Layer 6:      ████████████████████████████████████ 余烬/火星（Embers）
              飘散的橙色热粒子，逐渐熄灭
```

| 子类 | 物理特征 | 技术手段 |
|------|---------|---------|
| **闪光核心** | 极短时高亮，照亮周围环境 | Billboard 粒子 + 动态点光源 |
| **火球** | 翻滚的流体运动感 | 体积纹理序列 / Flipbook 动画 |
| **冲击波** | 空气折射扭曲 | 屏幕空间扭曲（Distortion）Shader |
| **碎片飞溅** | 受重力和碰撞影响的刚体 | 物理粒子 + 碰撞检测 |
| **烟尘** | 流体上升、扩散、消散 | 预渲染流体模拟 / 体积纹理 |
| **余烬** | 随风飘散、逐渐熄灭 | GPU 粒子 + 光照衰减 |

### 2.2 枪械特效

| 子类 | 说明 | 技术手段 |
|------|------|---------|
| **枪口焰（Muzzle Flash）** | 每次射击瞬间的火焰喷射 | 多层 Billboard + 动态光源（照亮枪手面部和周围环境） |
| **抛壳（Shell Ejection）** | 弹壳从枪膛弹出并落地 | 刚体物理粒子 + 金属碰撞音效 |
| **示踪弹（Tracer）** | 子弹飞行时的光轨迹 | 拉伸发光粒子 + 运动模糊 |
| **弹孔（Bullet Impact）** | 子弹击中表面的痕迹 | 贴花（Decal）系统 + 飞溅碎屑粒子 |
| **弹道扰动** | 子弹飞过时的空气扰动 | 微型屏幕扭曲效果 |

### 2.3 烟雾类

| 子类 | 说明 | 技术手段 |
|------|------|---------|
| **爆炸烟尘** | 爆炸后的浓烟 | 预渲染体积烟雾（SIGGRAPH 2014 论文） |
| **烟雾弹** | 战术遮蔽烟幕 | 实时 GPU 粒子系统 + 光照穿透 |
| **枪烟/硝烟** | 射击后残留的薄烟 | 低密度半透明粒子 |
| **废墟灰尘** | 建筑坍塌扬起的尘雾 | 体积雾 + 粒子混合 |
| **引擎/排气** | 载具排放的废气 | 温度驱动的上升粒子流 |

### 2.4 火焰类

| 子类 | 说明 | 技术手段 |
|------|------|---------|
| **燃烧火焰** | 持续燃烧的火焰（建筑、车辆） | Flipbook 动画 + 动态光源 + 热扭曲 |
| **白磷弹效果** | 白磷弹的剧烈燃烧和白烟 | 高亮粒子 + 大量浓白烟雾 |
| **燃烧瓶/火焰弹** | 地面燃烧的蔓延火焰 | 程序化火焰扩散 + 地表烧焦贴花 |
| **火星/飞火** | 爆炸或燃烧中飘散的火星 | GPU 小粒子 + 光照发射 |

### 2.5 水体与液体

| 子类 | 说明 | 技术手段 |
|------|------|---------|
| **水花飞溅** | 子弹入水、人物涉水 | 粒子喷射 + 涟漪贴花 |
| **雨滴** | 下雨天的大量雨滴 | GPU 粒子 + 地面涟漪 + 屏幕水滴 |
| **血液飞溅** | 命中敌人时的血雾 | 方向性粒子喷射 + 表面血迹贴花 |
| **水面波纹** | 水面受扰动产生的波纹 | 法线贴图动态扰动 |

### 2.6 环境粒子

| 子类 | 说明 | 技术手段 |
|------|------|---------|
| **灰尘漂浮** | 室内/废墟中的微粒漂浮 | 低速 GPU 粒子 + 体积光照穿透 |
| **雪花** | 雪天场景的降雪 | 风力驱动 GPU 粒子 |
| **树叶/纸片** | 爆炸或风力吹起的碎片 | 物理驱动旋转粒子 |
| **火山灰/沙尘** | 沙漠或火山场景 | 密集环境粒子 + 能见度衰减 |
| **萤火虫/光点** | 夜间场景的微光 | 缓慢飘动的发光粒子 |

## 三、材质与表面特效

### 3.1 PBR 物理材质系统

| 子类 | 说明 | CG 表现 |
|------|------|---------|
| **金属材质** | 金属度+粗糙度控制 | 枪械的磨损金属、弹壳的黄铜光泽 |
| **非金属材质** | 介电质标准 BRDF | 混凝土、木材、布料的真实质感 |
| **多层材质** | 表面层+基底层的复合计算 | 磨损油漆下露出金属、冰层下的地面 |
| **薄膜干涉** | 薄膜层对光波长的干涉 | 油渍彩虹色、枪管发蓝工艺 |
| **次表面散射（SSS）** | 光在半透明材质内部散射 | 人耳透光红色、蜡烛、树叶透光 |

### 3.2 动态表面效果

| 子类 | 说明 | 技术手段 |
|------|------|---------|
| **湿润效果** | 雨中/涉水后的湿润表面 | 降低粗糙度 + 增强反射 + 表面水膜 |
| **结霜/冰冻** | 寒冷场景的表面结霜 | 叠加霜冻法线 + 提高粗糙度 |
| **灼烧/烧焦** | 火焰/爆炸后的表面烧黑 | 程序化贴花 + 颜色渐变 |
| **弹痕/损伤** | 子弹/爆炸对表面的破坏标记 | 贴花系统（Decal System） |
| **动态污垢** | 角色/载具在泥地行走后的污渍累积 | 顶点色或程序化遮罩驱动的混合材质 |

### 3.3 特殊光学效果

| 子类 | 说明 | CG 表现 |
|------|------|---------|
| **热浪扭曲（Heat Haze）** | 高温表面上方的空气折射 | 沙漠远景扭曲、爆炸后热空气 |
| **镜面高光闪烁** | 粗糙表面的微观高光变化 | 沙粒/碎玻璃在阳光下的闪烁 |
| **各向异性反射** | 拉丝金属/头发的定向高光 | 枪管拉丝纹理、角色头发高光 |

## 四、破坏与物理特效

### 4.1 建筑破坏（SIGGRAPH 2014 论文技术）

| 阶段 | 说明 | 技术手段 |
|------|------|---------|
| **预断裂（Pre-fracture）** | 离线计算物体的断裂碎片分布 | Voronoi 碎片生成 |
| **触发破坏** | 受力超过阈值时触发断裂 | 物理碰撞检测 + 伤害系统 |
| **碎片飞散** | 碎块获得初速度飞出 | 刚体物理模拟 |
| **二级破坏** | 碎块与其他物体碰撞产生连锁反应 | 碎块间碰撞 + 二次碎裂 |
| **残骸沉降** | 碎块最终落地堆积 | 重力 + 摩擦 + 休眠检测 |

### 4.2 载具破坏

| 子类 | 说明 | CG 表现 |
|------|------|---------|
| **车辆变形** | 金属面板弯曲、凹陷 | 变形网格（Morph Target） |
| **部件脱落** | 车门、轮胎、保险杠飞出 | 断裂连接点 + 刚体物理 |
| **爆炸翻滚** | 车辆被炸翻 | 整体刚体旋转 + 部件分离 |
| **燃烧残骸** | 持续燃烧的残骸 | 火焰粒子 + 烟雾 + 金属变色 |

### 4.3 地形破坏

| 子类 | 说明 | CG 表现 |
|------|------|---------|
| **弹坑** | 爆炸在地面留下的坑洞 | 地形网格变形 + 泥土贴花 |
| **壕沟/战壕损伤** | 连续爆炸形成的地形改变 | 地形高度图修改 |
| **植被破坏** | 树木折断、灌木被炸飞 | 预断裂模型 + 物理碎片 |

### 4.4 可破坏物件

| 子类 | 说明 | CG 表现 |
|------|------|---------|
| **玻璃碎裂** | 窗户/车窗的碎裂模式 | 裂纹贴图 → 碎片粒子化 |
| **木质家具** | 桌椅/门板的碎裂 | 预断裂木块 + 木屑粒子 |
| **金属栏杆/围栏** | 弯曲变形或断裂 | 程序化弯曲 + 断裂检测 |

## 五、后处理特效（Post-Processing）

后处理特效是在场景渲染完成后，对整个画面进行的全屏处理。

### 5.1 抗锯齿

| 技术 | 说明 | CG 应用 |
|------|------|---------|
| **SMAA T2X** | 时间性子像素形态抗锯齿 | 消除边缘锯齿 + 消除闪烁 |
| **Filmic SMAA T2X** | 带电影感柔化的 SMAA | CG 过场动画的电影质感 |
| **TAA（时间性抗锯齿）** | 利用前帧信息平滑当前帧 | 预渲染 CG 的最终输出 |

### 5.2 景深（Depth of Field）

| 子类 | 说明 | CG 应用 |
|------|------|---------|
| **散景虚化（Bokeh DOF）** | 模拟物理镜头的焦外虚化效果 | 角色特写时的背景虚化 |
| **倾移景深** | 模拟微缩模型感的景深 | 战术地图/战场俯瞰 |
| **动态对焦** | 焦点随叙事自动变换 | 从前景角色转到背景爆炸 |

### 5.3 运动模糊（Motion Blur）

| 子类 | 说明 | CG 应用 |
|------|------|---------|
| **摄像机运动模糊** | 镜头移动产生的全屏模糊 | 高速追车场景、直升机快速平移 |
| **物体运动模糊** | 快速运动物体的拖尾效果 | 飞行中的子弹、挥动的拳头、飞驰的车辆 |
| **径向模糊** | 从中心向外的辐射状模糊 | 爆炸冲击波视角、加速冲刺效果 |

### 5.4 色彩处理

| 子类 | 说明 | CG 应用 |
|------|------|---------|
| **颜色分级（Color Grading）** | 调整整体色调/对比度/饱和度 | 每个场景独特的色彩氛围设定 |
| **LUT 调色** | 使用查找表进行电影级调色 | 冷战冰冷蓝调 vs 中东暖黄沙漠调 |
| **色差（Chromatic Aberration）** | 镜头边缘的 RGB 分离 | 模拟真实摄影机镜头缺陷 |
| **暗角（Vignette）** | 画面四角渐暗 | 聚焦画面中心、增加戏剧性 |

### 5.5 全屏特殊效果

| 子类 | 说明 | CG 应用 |
|------|------|---------|
| **闪光弹致盲** | 全屏高亮白化 → 逐渐恢复 | 被闪光弹命中的角色视角 |
| **核爆/EMP 脉冲** | 画面剧烈过曝 + 电子干扰 | MW 系列核弹场景 |
| **受伤效果** | 画面边缘泛红 + 模糊 | 角色受到伤害时的视觉反馈 |
| **水下效果** | 色调偏蓝绿 + 焦散光斑 | 潜水/涉水场景 |
| **夜视仪** | 绿色调 + 噪点 + 光晕 | 夜间渗透任务 |

## 六、角色特效

### 6.1 面部与表情

| 子类 | 说明 | 技术手段 |

| **皮肤渲染** | 皮肤的半透明质感 | 次表面散射（SSS）Profile |
| **泪水/汗水** | 角色情感表达 | 表面流体粒子 + 高反射材质 |

### 6.2 毛发与发型

| 子类 | 说明 | 技术手段 |
|------|------|---------|

| **胡须** | 面部毛发的精细表现 | Hair Card + 多层半透明排序 |
| **毛发修饰（Grooming）** | CG 预告片中的高精度毛发 | Houdini/XGen 离线修饰 |

### 6.3 布料与服装

| 子类 | 说明 | 技术手段 |

| **吉利服/伪装** | 丛林/沙漠伪装服的碎布条飘动 | 骨骼链 + 物理抖动 |
| **背包/装备** | 装备随运动晃动 | 辅助骨骼物理 |
| **旗帜/布帘** | 风中飘扬的旗帜 | 实时布料 + 不透明度材质 |

### 6.4 受伤与状态效果

| 子类 | 说明 | CG 表现 |
|------|------|---------|
| **伤口渲染** | 枪伤、弹片伤口 | 动态贴花 + SSS 变化 |
| **灰尘/血污** | 战斗中角色逐渐脏污 | 程序化污垢遮罩累积 |
| **湿润效果** | 雨中/涉水后全身湿透 | 粗糙度降低 + 高光增强 |

## 七、镜头语言特效

CG 过场动画大量使用电影级镜头语言特效来增强叙事：

### 7.1 时间操控

| 子类 | 说明 | CG 应用 |
|------|------|---------|
