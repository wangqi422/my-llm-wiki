## 1. 概述

使命召唤（Call of Duty）系列是游戏行业中 CG 过场动画和宣传片制作水准最高的 IP 之一。从 2003 年初代至今 20 余年间，其 CG 视频经历了从预渲染 FMV 到实时引擎内过场、再到混合媒介 CG 预告片的完整演进。

**CG 视频的类型分布：**

| 类型 | 说明 | 典型案例 |
|------|------|---------|
| **预渲染 CG 预告片** | 由外部 CG 公司制作，用于游戏宣发 | Blur Studio / Platige Image 制作的发售预告 |
| **引擎内过场动画** | 使用游戏引擎实时渲染，与游戏画面无缝衔接 | MW2019 / MW2 的全程实时过场 |
| **混合媒介叙事 CG** | 融合 3D CG、档案影像、动态图形 | Territory Studio 为 BO6 制作的战前简报动画 |
| **引擎内 CG（增强版）** | 游戏引擎渲染但使用更高规格的模型和光照 | COD: WWII 的诺曼底登陆序幕 |

## 3. 核心特效技术分析

### 3.1 动作捕捉与面部捕捉

#### 全身动作捕捉（Performance Capture）

**技术方案：OptiTrack 光学动捕系统**

- **设备**：Infinity Ward 使用虚拟动点（NOKOV / OptiTrack）光学动捕系统
- **流程**：演员穿戴反光标记点紧身衣，在专用 PCAP（Performance Capture）摄影棚中进行全身表演
- **同步捕获**：同一阶段同时捕获动作、表情、语音，确保精确同步
- **军事真实性**：聘请真实特种兵人员进行动捕，确保战术动作的专业性
- **典型案例**：《现代战争》(2019) 中，Barry（特种兵顾问）与扮演队友的演员共同完成了全部过场动捕

#### 面部捕捉（DI4D PURE4D 系统）

**《现代战争 II》的面部动画管线是行业标杆：**

| 参数 | 规格 |
|------|------|
| **HMC 设备** | 8 台 Technoprops HMC（头戴式面部捕捉） |
| **摄像头** | 每台 9 个 1200 万像素摄像头，30fps |
| **日数据量** | 每天 >1 TB HMC 素材 |
| **主要演员** | 14 位，100+ 种姿势数据 |
| **处理技术** | DI4D PURE4D —— 基于机器学习的 4D 面部数据处理 |

**AI 加速效果：**
- 传统手动清理 4D 面部数据需 **2 周**
- 使用 PURE4D 机器学习算法后仅需 **3 天**
- 能捕捉"中间态"微妙面部表情（如半眨眼、嘴角微颤等）

**关键创新点：**
1. **实时评估**：动画基于演员实际表演实时评估，非后期调整
2. **减少主观分歧**：自动化减少了表演者、动画师、总监间的反复修改
3. **数字替身**：游戏角色实质是真人特种兵的"数字分身"（Digital Double）

### 3.2 摄影测量（Photogrammetry）

**《现代战争》(2019) 将摄影测量技术推向新高度：**

#### 环境扫描
- **微距级纹理**：达到 **64 像素/英寸** 的细节密度（前代仅 16-32 px/in）
- **无人机航拍扫描**：用于海岸线、沙漠、山脉等大场景
- **废墟采集**：团队实地拍摄采石场、废弃建筑、路边碎片，直接导入引擎

#### 物体与载具扫描
- **全方位扫描**：如一辆完整的俄罗斯坦克，包括底部细节
- **道具做旧**：物理做旧处理（浸泡、磨损、风化），扫描后保持真实磨损感

#### 角色扫描
- **"肉袋"技术**：真人模仿尸体状态进行扫描，捕捉真实重力效果
- **服装做旧**：角色服装经过实际穿戴磨损（徒步穿鞋、泳池浸泡、工具打磨）后扫描

#### 远景处理
- 支持 **12 倍镜** 级别的远距离细节
- 近景到远景的视觉一致性通过多级 LOD + 摄影测量数据保证

### 3.3 PBR 物理渲染技术

使命召唤系列是游戏行业 PBR（Physically Based Rendering）技术的先驱和标杆。

#### 历史贡献

| 时间节点 | 里程碑 |
|---------|--------|
| 2011 | 《黑色行动》在 SIGGRAPH 发表 PBR 技术分享，为行业提供信心背书 |
| 2014 | IW6 引擎（幽灵）全面支持 DX11 PBR 管线 |
| 2018 | Sledgehammer 在 SIGGRAPH 发表《使命召唤：二战》PBR 深度论文 |
| 2019 | IW 8.0 引擎（现代战争）实现照片级 PBR |
| 2025+ | IW 9.0 引擎支持 PBR 2.0 + SDF-GI |

#### 核心技术细节（SIGGRAPH 2018 论文）

**1. Normal & Gloss Mipmapping（法线与光泽度多级渐远）**
- **问题**：远距离观察时法线细节因 Mipmapping 丢失，表面看起来过于光滑
- **解决方案**：将法线贴图视为光泽度的一部分，细节丢失时转移到光泽度的 Mipmap 中

**2. Material Surface Occlusion（材质表面遮挡）**
- 利用法线贴图生成高度图 → AO 和自阴影信息
- 改进传统 AO：考虑邻近材质的间接反射，遮挡区域不再简单变黑

**3. Multiscattering Diffuse BRDF（多重散射漫反射）**
- 解决传统模型的能量不守恒问题
- 根据高光反射能量调整漫反射光照量
- 引入微表面间的多重散射能量计算

**4. Thin Film Layer（薄膜层）**
- 模拟油膜、涂层等薄膜对光波长的干涉效应
- 极大增强了枪械表面、车辆漆面等材质的真实感

**5. 统一着色模型（Unified Shading Model）**
- 向电影 CG 行业靠拢
- 支持多层材质计算（光的吸收与散射积累）
- 推动渲染管线从延迟渲染向前向渲染演进

### 3.4 粒子系统与破坏模拟

#### 破坏序列（SIGGRAPH 2014 —— COD: Ghosts）

《使命召唤：幽灵》发表了破坏模拟技术论文，涵盖：

| 技术 | 说明 |
|------|------|
| **断裂模拟（Fracture Simulation）** | 建筑物、车辆等可破坏物体的物理断裂计算 |
| **密集局部粒子模拟** | 破坏点周围的高密度碎片粒子系统 |
| **预渲染烟雾** | 将高质量流体模拟烟雾预渲染为体积纹理，实时播放 |
| **碎片物理** | 断裂碎块具有独立物理属性，与环境交互 |

#### 战场特效粒子系统

CG 和游戏内均使用的粒子特效类别：

- **爆炸系统**：多层粒子（火球核心 → 冲击波 → 碎片飞溅 → 烟尘扩散 → 余烬）
- **枪口火焰**：基于物理的枪口焰光照，影响周围环境光照
- **弹道轨迹**：示踪弹光效、子弹破风扰动
- **环境粒子**：灰尘漂浮、雨滴、雪花、火星飘落
- **破坏碎片**：混凝土飞溅、玻璃碎裂、木屑飞散

### 3.5 光照与全局光照系统

#### IW Engine 光照演进

| 引擎版本 | 光照特性 |
|---------|---------|
| IW 3.0 (COD4) | 静态光照贴图 + 动态点光源 |
| IW 5.0 (MW3) | 改进的流式光照 |
| IW 6.0 (Ghosts) | 完整 DX11，首次大幅提升环境光照 |
| IW 8.0 (MW2019) | **混合光线追踪**、实时光线追踪阴影、体积光照 |
| IW 9.0 (最新) | **SDF-GI**（有符号距离场全局光照）、PBR 2.0 |

#### 光线追踪在 CG 中的应用

《现代战争》(2019) 引擎首次引入实时光线追踪：
- **光追阴影**：柔和且物理正确的阴影投射
- **光追反射**：金属、水面、玻璃的精确镜面反射
- **光追环境光遮蔽**：替代传统 SSAO，精度显著提升
- **体积雾光**：支持光线穿透烟雾/灰尘的散射效果

### 3.6 动态图形与混合媒介合成

#### Territory Studio 的技术革新（BO6）

Territory Studio 为《黑色行动 6》创造了全新的 CG 叙事形式：

**解构主义设计语言：**
- 打破整体再重组，制造视觉的"强烈感"和"混乱感"
- 对应 90 年代初（游戏设定年代）技术飞速发展的时代背景
- 粗犷排版强化叙事的紧张、不和谐与不对称

**"心眼"（Mind's Eye）动画概念：**
- 模拟特工在规划任务时的心理可视化过程
- 地图随镜头飞行时动态生成
- 档案内容围绕关键信息自动重组
- 照片变形为叠加信息层

**全手工资产创建：**
- 墨迹、手写涂鸦、揉皱黑色胶带、拳头涂抹的记号笔痕迹
- 数百种不同纹理扫描后集成到动画时间轴
- CG、档案影像素材与粗糙排版的多层合成

**灯光叙事：**
- 摄影机保持黑暗且不断移动
- 灯光作为场景间动势的关键因素
- 光线引导观众穿越叙事流程

**项目规模：14+ 分钟定制动画**

| 类别 | 数量 |
|------|------|
| 标题序列 | 1 个 |
| 战前动画 | 7 个 |
| 安全屋动画 | 5 个 |
| 任务中动画 | 3 个 |

#### Platige Image 的 2D/3D 混合技术（黑色行动冷战）

**角色模型升级：**
- 游戏内低模 → 数百万多边形高精度 CG 模型
- 完整的毛发修饰、角色特效模拟

**双重视觉氛围：**
- 第一部分：黑暗灯光 + 红色点缀 → 愤怒与残酷
- 第二部分：霓虹灯 + 80 年代购物中心 → 时代感

**"照片拼贴"序列：**
- 3D 动画与 2D 镜头交织
- 回顾性叙事中的情感增强手法

## 4. 各代作品 CG 特效亮点

### 4.1 《使命召唤：幽灵》(2013) — Mill+ 的电影级 CG

**制作方**：Mill+（The Mill 旗下概念设计与动画部门）
**导演**：Paul Mitchell（曾参与《钢铁侠 2》）
**团队**：洛杉矶 + 伦敦双城协作

**技术亮点：**
- 好莱坞级别的视效合成流程
- 大规模破坏场景的断裂模拟 + 粒子系统（配套 SIGGRAPH 论文）
- 由 Infinity Ward 的 Cinematics 团队主导，Mill+ 负责执行

### 4.2 《使命召唤：现代战争》(2019) — 引擎革命

**IW 8.0 引擎的突破性升级：**
- **全程实时过场动画**：不再使用预渲染 CG，所有过场均在引擎内实时渲染
- **摄影测量管线**：64px/in 极致细节密度
- **混合光线追踪**：首次在 COD 系列中引入
- **PBR 全面升级**：照片级材质表现

**Digital Foundry 评价**："系列迄今最先进的视觉效果，展示了 COD 渲染技术的根本性革新"

### 4.3 《使命召唤：现代战争 II》(2022) — AI 面捕突破

**核心创新：DI4D PURE4D 面部动画管线**
- 8 台 HMC 设备，9 个 1200 万像素摄像头
- 机器学习将面部数据处理从 2 周压缩至 3 天
- 14 位主演的数字替身，真实特种兵扫描

**面部动画管线（B站/SIGGRAPH 技术分享）：**
- 角色 RIG 制作流程
- 面部捕捉数据 → RIG 的自动映射
- 数字分身（Digital Double）全流程

### 4.4 《使命召唤：黑色行动 冷战》(2020) — Platige Image

**制作团队（核心）：**
- 导演：Damian Nenow
- CG 主管：Hubert Zegardło
- 艺术总监：Karol Klonowski

**技术特色：**
- 游戏资产 → 数百万面高模的重构管线
- 双重视觉氛围设计（暴力黑暗 vs 80 年代霓虹）
- 2D/3D 混合"照片拼贴"序列
- 从分镜 → 动态预览 → 预可视化的完整电影制作流程

### 4.5 《使命召唤：黑色行动 6》(2024) — Territory Studio

**制作周期**：超过 2 年
**总时长**：14+ 分钟定制动画
**创意总监**：Ryan Jefferson Hays

**技术特色：**
- 解构主义视觉语言
- "心眼"叙事动画概念
- 全手工物理资产（墨迹/胶带/涂鸦）扫描 + 数字集成
- 运镜与灯光的叙事性驱动设计
- 混合媒介（CG + 档案影像 + 动态排版）分层合成

## 5. 引擎技术演进

### IW Engine 完整演进时间线

| 引擎版本 | 首发游戏 | 年份 | 核心技术升级 |
|---------|---------|------|-------------|
| id Tech 3 (改) | COD (初代) | 2003 | 基础 OpenGL 渲染 |
| IW 3.0 | COD4: 现代战争 | 2007 | 首个自研引擎，显著提升画面和特效 |
| IW 4.0 | COD: 现代战争 2 | 2009 | 改进的流式技术、更好的光照 |
| IW 5.0 | COD: 现代战争 3 | 2011 | 流式技术与音频效果提升 |
| IW 6.0 (次时代引擎) | COD: 幽灵 | 2013 | 完整 DX11、破坏系统、Sub-D 细分曲面 |
| IW 7.0 | COD: 无限战争 | 2016 | 改进粒子系统 |
| IW 8.0 | COD: 现代战争 (2019) | 2019 | **革命性升级**：摄影测量、混合光追、PBR 升级、全实时过场 |
| IW 9.0 | 最新系列 | 2025+ | SDF-GI 全局光照、PBR 2.0、AI 行为树 |

### Treyarch NGL 引擎

- 基于 id Tech 3 独立分支
- 用于黑色行动系列
- 预渲染 CG 过场 + 引擎内过场混合使用

## 7. 总结与行业影响

### 使命召唤 CG 特效的行业贡献

1. **推动 PBR 行业标准化**：2011 年《黑色行动》的 SIGGRAPH 分享为 PBR 在游戏行业的大规模采用提供了关键信心背书，从此 PBR 成为 3A 游戏的标配
2. **实时过场动画标杆**：《现代战争》(2019) 证明了游戏引擎可以实现与预渲染 CG 媲美的视觉质量，推动了整个行业向实时过场转型
3. **AI 面部捕捉先驱**：DI4D PURE4D 系统在 MW2 中的应用展示了机器学习在角色动画管线中的巨大潜力
4. **摄影测量工业化**：COD 团队将摄影测量从实验技术推进为工业化生产流程
5. **混合媒介叙事创新**：Territory Studio 为 BO6 创造的解构主义混合媒介 CG 为游戏叙事开辟了新方向

### 技术发展趋势

| 方向 | 当前状态 | 未来展望 |
|------|---------|---------|
| **实时渲染** | IW 9.0 引擎已支持实时全局光照 | 全面取代预渲染 CG |
| **AI 驱动动画** | PURE4D 机器学习面部处理 | AI 全身动作生成、风格迁移 |
| **光线追踪** | 混合光追（部分场景） | 全路径追踪实时渲染 |
| **虚拟制作** | 动捕棚 + 引擎预览 | LED Volume 虚拟拍摄一体化 |
| **程序化内容** | 摄影测量为主 | 程序化生成 + AI 纹理合成 |
