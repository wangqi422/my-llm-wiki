# CODM 历代版本视觉指引（Visual Guide）索引

> 来源：团队内部 ArtHub / 企微文档 | 时间跨度：S27.1 – S36.2

## 概述

本文档索引了使命召唤手游（CODM）从 S27.1 到 S36.2 的所有版本视觉指引（Visual Guide），涵盖通用版本规范、IP 联动合作指引等。视觉指引是每个赛季宣发设计的核心规范文档，定义了该赛季的视觉风格、配色、构图规范等。

## 关键观察

1. **IP联动频率加速**：从 S28 开始出现 IP 联动指引，到 S32-S35 几乎每个版本都有联动
2. **命名规范演变**：早期称"版本规范"，后期统一为"版本视觉指引/GUIDE"
3. **存储平台迁移**：早期使用 `arthub.qq.com/j3_uxdc/pan`，部分使用 `arthub.woa.com/codm/pan`
4. **部分版本链接缺失**：S29.2 到 S35.2 多个版本缺少 ArtHub 链接，可能需要补充
5. **36.2 克苏鲁**为最新版本，也有企微文档幻灯片版本（`doc.weixin.qq.com/slide/`）
